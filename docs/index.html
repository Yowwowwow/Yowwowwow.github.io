s-byel s-bye/am

Web version    Read me

You can also call it Spell Spam. A game where you use eight spells to eliminate enemies and try to survive as long as possible. Made in 2021-2022. (However, the loading screen was changed in 2024.)


gèn srruài chí

Web version

A chess-like board game with many additional features, including online multiplayer support. Made in 2020-2021.


yù túl chí

Web version

A board game where you connect pieces of the same color to remove them and gain points. When all pieces are removed, the players with the most points win. Made in 2023. Note: the in-game tutorial says a player loses if they have ≤7 points when they end their turn; this is wrong. A player only loses if they have ≤-7 points when they end their turn.


jiǎng n̄ chí

Web version

A game where you connect pieces on a hexagonal board to remove them. It's like a sandbox; you can customize the rules of the game. Made in 2023-2024.


Climbing Xiangqi

Web version    How to play

A variant of xiangqi (Chinese chess), where pieces can stand on each other! Make your opponent run out of moves to win! Made in 2024.
