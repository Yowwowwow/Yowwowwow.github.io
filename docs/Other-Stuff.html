艮牌

Web version    How to play

A simplified and digitalized (and very old) version of a physical card game I made, called 艮牌 (gèn pái). It is very buggy, be warned 💀 If the game freezes for too long, it's probably softlocked and you should just refresh. There's a known bug where you or the computer plays multiple cards at once and/or the cards not displaying properly; if it occurs, you can play on or refresh.

憨兒的迷宮冒險3D

Download (Extract the zip file and run 憨兒的迷宮冒險3D.exe to play)

A goofy 3D maze game (with clumsy controls)!!! Use ↑←→↓ to move, space to jump, WASD to pan the camera, Z to attack. Find the hidden yellow star to win! Avoid the red enemy; if you touch it for too long, you lose! However, if you attack while touching it, it will be teleported away.

艮艮寸球

Download (apk file, run it on Android to install the game)

A funny little game where you control the green ball with the gyroscope sensor! If the green ball touches the red ball, you win! But if it touches the edge or the black obstacles too many times, you lose!
